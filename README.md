# Бачурин Никита ИТС-2 Лабораторная №4

# Задание 1

## Задача 11

### Текст задачи

Дерево содержит целые числа. К каждому числу приписать заданную цифру в

начало

### Алгоритм решения

1.Подключается библиотека для ввода, вывода и генерации случайных чисел

2.Описывается структура дерева: узел с числом и двумя поддеревьями или пустое значение

3.Описывается правило добавления цифры в начало числа

4.Проверяется знак числа

5.Если число положительное - цифра приписывается слева

6.Если число отрицательное - убирается минус, добавляется цифра и возвращается минус

7.Описывается обход дерева с применением действия к каждому элементу

8.Если дерево пустое - возвращается пустое дерево

9.Если есть узел - обрабатывается его значение

10.Рекурсивно обрабатывается левое поддерево

11.Рекурсивно обрабатывается правое поддерево

12.Создаётся новый узел из полученных результатов

13.Описывается функция применения добавления цифры ко всему дереву

14.Для этого используется обход дерева

15.Описывается вставка числа в дерево

16.Если дерево пустое - создаётся новый узел

17.Если число меньше текущего - переход в левое поддерево

18.Иначе - переход в правое поддерево

19.Повторяется процесс, пока не найдётся место

20.Описывается вывод дерева

21.Если дерево пустое - ничего не выводится

22.Сначала выводится правое поддерево

23.Затем выводится текущее число с отступом

24.Затем выводится левое поддерево

25.Описывается ввод одной цифры

26.Считывается строка с клавиатуры

27.Проверяется, что введена одна цифра

28.Если корректно - преобразуется в число

29.Если нет - выводится сообщение об ошибке

30.Повторяется ввод до корректного значения

31.Запускается основная часть программы

32.Создаётся генератор случайных чисел

33.Запрашивается количество чисел

34.Считывается введённое значение

35.Генерируется список случайных чисел

36.Выводится список чисел

37.Создаётся дерево из списка

38.Выводится исходное дерево

39.Запрашивается цифра для добавления

40.Считывается корректная цифра

41.Создаётся новое дерево с добавленной цифрой

42.Выводится новое дерево

43.Программа завершает работу

# Задание 2

## Задача 11

### Текст задачи

Сколько в дереве, содержащем натуральные числа, элементов заданной

значности (например, трёхзначных)? Значность задается

### Алгоритм решения

1.Подключается библиотека для ввода, вывода и работы с числами

2.Описывается структура дерева с узлом и пустым значением

3.Описывается правило добавления числа в дерево

4.Если дерево пустое - создаётся новый узел

5.Если число меньше текущего - переход в левую часть

6.Иначе - переход в правую часть

7.Повторяется процесс, пока не найдётся место для вставки

8.Описывается вывод дерева на экран

9.Если дерево пустое - ничего не выводится

10.Сначала выводится правая часть дерева

11.Затем выводится текущее число с отступом

12.Затем выводится левая часть дерева

13.Описывается обход дерева с накоплением результата

14.Если дерево пустое - возвращается накопленный результат

15.Сначала обрабатывается левая часть дерева

16.Затем обрабатывается текущее число

17.Затем обрабатывается правая часть дерева

18.Результат постепенно накапливается при обходе

19.Описывается правило проверки количества цифр числа

20.Число переводится в текст

21.Определяется длина текста

22.Сравнивается длина с заданным значением

23.Если совпадает - увеличивается счётчик

24.Если нет - счётчик не изменяется

25.Описывается ввод натурального числа

26.Считывается строка с клавиатуры

27.Проверяется, что введено число больше нуля

28.Если корректно - используется дальше

29.Если нет - выводится сообщение об ошибке

30.Повторяется ввод до корректного значения

31.Запускается основная часть программы

32.Создаётся генератор случайных чисел

33.Запрашивается количество чисел

34.Считывается введённое значение

35.Генерируется список случайных натуральных чисел

36.Выводится список чисел

37.Создаётся дерево из списка

38.Выводится исходное дерево

39.Запрашивается количество цифр для проверки

40.Считывается введённое значение

41.Запускается обход дерева с подсчётом

42.Для каждого числа проверяется количество цифр

43.Подсчитывается общее количество подходящих чисел

44.Выводится результат подсчёта

45.Программа завершает работу
